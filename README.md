# Pydantic Advanced Tutorial

This is the repository for my tutorial on [YouTube](https://youtu.be/mmYYXW9FWDI).

In this tutorial, you will learn about the following features of [Pydantic](https://docs.pydantic.dev):
- default factory
- aliases
- computed fields
- field validators
- model validators
